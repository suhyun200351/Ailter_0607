Ailter

The design system for Ailter — “AI Filter”, a mobile service that helps teens judge the trustworthiness of AI-generated social content. Built on Pretendard, a warm red brand, and a calm gray scale.

10 type styles 19 core colors 13 component families Platform · iOS mobile Type · Pretendard Variable

Foundations

Color

A single red Primary ramp carries the brand, balanced by a neutral gray scale for surfaces and text. Three semantic colors encode content trust — the heart of the Ailter experience.

Foundations

Typography

Pretendard Variable is the single UI typeface — it covers Korean and Latin with even rhythm. The logo wordmark is set in Wanted Sans ExtraBlack. Every style uses a tight −2% letter-spacing tuned for Hangul.

Foundations

Spacing & Radius

A 4px base grid governs spacing. Corner radii step from sharp tags to soft cards, with a full pill for toggles and chips.
